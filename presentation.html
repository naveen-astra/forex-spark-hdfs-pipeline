Batch Pipeline

Spark preprocessing

SparkHDFSPreprocess.scala reads 45 raw CSVs, applies window functions for moving averages and volatility, then writes enriched output to spark_output/.

Step 01
Ingest
Read CSV from raw2/ with schema inference. 15 pairs × 3 timeframes.
Step 02
Window Functions
MA5, MA20 via avg().over(Window). Volatility5 via rolling stddev.
Step 03
Derived Columns
Range, Change, ChangePct, Gain, Loss for RSI components.
Step 04
Output
Write to spark_output/ as CSV. 10.8 GB total after feature expansion.

Key Spark operations

// Window functions for distributed moving average val window5 = Window.orderBy("Date").rowsBetween(-4, 0) val window20 = Window.orderBy("Date").rowsBetween(-19, 0) df.withColumn("MA5", avg("Close").over(window5)) .withColumn("MA20", avg("Close").over(window20)) .withColumn("Volatility5", stddev("Close").over(window5)) .withColumn("Gain", when(col("Change") > 0, col("Change")).otherwise(0))

Live Queries

Streaming queries

forex_tick_stream

Tick-level enriched data with MA crossover signals (BUY/SELL/HOLD), momentum classification, and volatility alerts.

Trigger: every 10 seconds

forex_window_stats

5-minute sliding window aggregation — avg/min/max close, price std deviation, average volatility per pair.

Trigger: every 30 seconds

forex_live_mem

In-memory queryable table with all enriched columns. Accessible via Spark SQL at runtime.

Trigger: every 10 seconds
